solusi untuk memperbaiki chrome

1. pake browser

Win + R, ketik iexplore.exe atau control inetcpl.cpl
klik tanda tanya di pojok kanan atas terus masuk tab general dan klik language/accessibility
kalo udah kebuka install windows 11 ISO


2. paksa berhenti
(di mode normal)
command promt run as administrator ketik chkdsk C: /f /r
setelah muncul pesan ketik Y
terus restart dan tunggu (layar akan menjadi hitam)


3. jalanin SFC di mode mormal

buka cmd administrator terus ketik sfc /scannow

PLAN 1
(di mode normal)
command promt run as administrator ketik chkdsk C: /f /r
setelah muncul pesan ketik Y
terus restart dan tunggu (layar akan menjadi hitam)
setelah masuk dekstop jalanin di cmd administrator --> sfc /scannow

PLAN 2
hubungkan dengan wi fi 
di cmd administrator run --> DISM.exe /Online /Cleanup-image /Restorehealth
stelah selesai restart laptop

PLAN 3
download file Windows 11 ISO terus klik kanan pilih mount dan lakuin instruksinya

PLAN 4 (SOLUSI TERAKIR)

Coba dulu atur ulang BIOS
Matiin laptop terus nyalain
pas layar masih hitam spam tombol F2
di menu BIOS cari opsi load setup defaults atau reset to default
buka setting (Win + i)
masuk ke system > recovery
pada opsi reset this PC klik tombol reset PC
pilih opsi keep my files atau remove everything
